# Achilles - MSG Info System Specification

## Project Overview

**Achilles** is an information system for tracking use of MSG (monosodium glutamate) and related additives in food products. It provides a TUI (terminal user interface), Tkinter GUI, and PHP web interface for managing a database of food items, their ingredients, and associated symptoms.

## Architecture

### Middleware Pattern

The system uses a shared middleware architecture to avoid code duplication:

```
┌─────────────────────────────────────────────────────────────┐
│                    Business Logic (achilles)                │
│  ┌─────────────────────────────────────────────────────────┐│
│  │  fooditem.py    - Data model and DB operations          ││
│  │  ui_schema.py   - Declarative UI metadata                ││
│  │  conf.py        - Runtime configuration                  ││
│  └─────────────────────────────────────────────────────────┘│
│                           ▲                                  │
│                           │                                  │
├───────────────────────────┼─────────────────────────────────┤
│                    MIDDLEWARE LAYER                          │
│  ┌──────────────┬──────────────┬──────────────┬────────────┐ │
│  │   TUI        │    WWW       │   Tkinter    │   App     │ │
│  │  (bbsengine6)│   (PHP)      │  (Python)    │ (Mobile)  │ │
│  └──────────────┴──────────────┴──────────────┴────────────┘ │
└─────────────────────────────────────────────────────────────┘
```

### ui_schema() Pattern

All UIs (TUI, Tkinter, WWW) use the same `ui_schema()` function to render forms:

```python
def ui_schema(args=None, context=None):
    return {
        "FoodItem": {
            "upc": {"type": "text", "label": "UPC Code"},
            "name": {"type": "text", "label": "Name", "required": True},
            "msgpresent": {"type": "bool", "label": "MSG Present?"},
            "qsr": {"type": "bool", "label": "Quick Service Restaurant?"},
            # ... more fields
        }
    }
```

## Database Schema

### Tables

- **achilles.__fooditem** - Food items with columns (not JSONB)
- **achilles.__purchase** - Purchase records
- **achilles.map_fooditem_additive** - Food item to additive mapping
- **achilles.map_fooditem_ingredient** - Food item to ingredient mapping

### Views

- **achilles.fooditem** - Public view of fooditems
- **achilles.purchase** - Public view of purchases
- **achilles.additive** - Food additives
- **achilles.ingredient** - Food ingredients
- **achilles.brand** - Brands
- **achilles.corp** - Corporations
- **achilles.manuf** - Manufacturers
- **achilles.vendor** - Vendors
- **achilles.merchant** - Merchants
- **achilles.symptom** - User-reported symptoms

### FoodItem Fields

| Field | Type | Description |
|-------|------|-------------|
| id | bigserial | Primary key |
| upc | text | UPC barcode |
| sku | text | Stock keeping unit |

| title | text | Display title |
| description | text | Description |
| brandid | bigint | FK to brand |
| manufid | bigint | FK to manufacturer |
